# Pamela Camacho — Portfolio

A single-page portfolio inspired by **My Neighbor Totoro** and **Spirited Away**: rolling green hills, golden-hour sky, drifting clouds, lantern-style accents, and soot-sprite details.

## What’s inside

- **About** — Tagline, location, and short bio
- **Experience** — Assembled (GTM), Darktrace, VMWP, Bureau Veritas
- **Education** — CSU Fullerton (CS), CSU Northridge
- **Skills & certifications** — CompTIA A+, IT, customer service, and more
- **Contact** — Link to your LinkedIn

## How to view

Open `index.html` in a browser (double-click or run):

```bash
open index.html
```

No build step or server required.
